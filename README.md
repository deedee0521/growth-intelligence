# MoeGo Growth Intelligence Dashboard

A comprehensive growth analytics dashboard for pet care businesses.

## Live Demo
[View Dashboard](https://mgdeedee.github.io/growth-intelligence/)

## Features
- **Growth Health Overview** - Executive scorecard with key metrics
- **Conversion Funnel** - Lead → Customer → Paid tracking
- **Channel Intelligence** - Performance by acquisition channel with Quality Score
- **Customer Value** - Segmentation (VIP, Regular, Light, Converted Only)
- **Cohort Analysis** - Monthly retention tracking
- **AI Insights** - Automated risk detection and opportunities
- **Location Portfolio** - Multi-location performance comparison

## Data Import
Upload CSV exports:
- Leads export
- Clients export
- Appointments export (optional)
- Transactions export (optional)
